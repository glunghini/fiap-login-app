## Descrição do Projeto
Este projeto consiste em uma tela de login desenvolvida para a FIAP utilizando React Native com Expo, TypeScript e estilização via StyleSheet e NativeWind. O foco principal é a entrega de uma interface funcional que respeite a identidade visual da instituição e ofereça recursos modernos de usabilidade.

### Ideia Principal
A aplicação foi estruturada para simular um fluxo de autenticação real, permitindo ao usuário escolher entre diferentes métodos de identificação e personalizar a visualização da interface.

### Requisitos Implementados
De acordo com os requisitos solicitados, o projeto utiliza os seguintes elementos:
* Texto: Rótulos e mensagens informativas.
* Input: Campos de entrada para e-mail, celular e senha.
* Botão: Ações de login e navegação.
* Imagem: Logotipo oficial da FIAP nos assets e na Splash Screen.
* Switch: Controle de permanência de login e alternância de tema.
* Estilização: Uso de StyleSheet para garantir a fidelidade visual.

### Funcionalidades
* Alternância de Tema: Botão toggle que altera entre modo claro e modo escuro.
* Abas de Acesso: Seleção entre entrada por E-mail ou Celular.
* Mascara de Dados: Formatação automática do campo de telefone para o padrão brasileiro.
* Splash Screen: Tela de carregamento personalizada com fundo preto e logo da FIAP.

* NativeWind

## Como Executar o Projeto
### Pré-requisitos
* Node.js instalado.
* Aplicativo Expo Go instalado no dispositivo móvel.

### Passo a Passo
1. Clone este repositório:
```bash
git clone [https://github.com/glunghini/fiap-login-app.git](https://github.com/glunghini/fiap-login-app.git)
```
2. Instale as dependências:
```bash
npm install
```
3.Inicie o servidor de desenvolvimento:
```bash
npx expo start -c
```
4.Utilize o aplicativo Expo Go para visualizar o projeto no dispositivo móvel.
